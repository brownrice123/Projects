# Projects

Coursework and projects from Drexel University.

## Contents

### Housing Price Prediction
[`FinalProjectCodeV3.ipynb`](FinalProjectCodeV3.ipynb)

Linear regression model predicting residential sale prices using the Ames Housing dataset (~80 features). Covers feature engineering for categorical variables, missing data handling, train/test splitting, and model evaluation via MAE, MSE, and RMSE.

**Tools:** Python · pandas · scikit-learn · matplotlib · seaborn

---

### Chronic Kidney Disease Case Study
[`casestudy.R`](casestudy.R)

Exploratory analysis of CKD clinical indicators. Covers missing data handling, correlation analysis, dummy encoding for categorical variables (race group, care source), linear regression modeling, and 3D PCA visualization.

**Tools:** R · pca3d
